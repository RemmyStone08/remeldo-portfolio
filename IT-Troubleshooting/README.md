Highlights:

## Troubleshooting method

Each case follows a structured process:
1. Identify the issue
2. Verify the symptoms
3. Test multiple possibilities
4. Rule out false causes
5. Determine the root cause
6. Apply or recommend the best solution
7. Record the outcome clearly
