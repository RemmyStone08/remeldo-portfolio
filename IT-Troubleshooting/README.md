# IT Troubleshooting Portfolio

This section documents real-world troubleshooting cases, focusing on diagnosis, testing, and resolution of hardware and software issues.

## Cases

### 1. RAID to AHCI Conversion (Dell Inspiron 15)
- Safely converted storage mode from RAID to AHCI
- Prevented boot failure using Safe Mode method
- Demonstrated BIOS and Windows boot troubleshooting

### 2. 100% Disk Usage Diagnosis (HP 15, RST + SSD)
- Diagnosed persistent 100% disk usage
- Identified limitation caused by Intel RST and a DRAM-less SSD
- Differentiated between SSD health and performance issues
- Provided both temporary optimizations and permanent solution

## Skills Highlighted

- Hardware and software troubleshooting
- BIOS configuration and limitations
- Storage controller management (RAID vs AHCI)
- Performance diagnostics
- Root cause analysis
- Client communication and solution recommendation

## Approach

Each case follows a structured method:
1. Identify the problem
2. Verify symptoms
3. Test multiple scenarios
4. Eliminate possible causes
5. Determine root cause
6. Provide solution (temporary and permanent where applicable)
